# Radio

Элемент, который позволяет выбрать только одно значение из нескольких.

При необходимости ознакомьтесь с [исходным кодом](https://github.com/2e32/react-radio) компонента.

## Начните работать

В storybook представлены материалы, позволяющие ознакомиться с функционалом компонента.

_Свойства (props)_:

- изучите список свойств, которые принимает компонент
- изменяйте значения свойств, чтобы увидеть какой эффект они дадут

_Использование (usage)_:

- ознакомьтесь с расширенным применением. Некоторые примеры могут носить практический характер
